# React ではじめる

React から Spacha を呼び出すことができます．

## インストール

npm から react-spacha をインストールします．

```
npm i react-spacha
```


## 呼び出す

<Tabs groupId="cview-types">
<TabItem value="html" label="HTML" default>

<div style={{width: "600px", marginBottom: "10px"}}>
    <Spacha
        price={200}
        message="Hello Spacha from React!"
    />
</div>

```jsx
import { Spacha } from "react-spacha";

<Spacha price={200} message="Hello Spacha from React!" />;
```

:::tip
`Spacha`コンポーネント(HTML 版)には`width`や`height`を指定する機能がありません．Spacha の幅が大きすぎる場合は，div タグで`width`を固定すると良いかもしれません．

```jsx {3,8}
import { Spacha } from "react-spacha";

<div style={{width: "600px"}}>
    {icon.conplete ? <Spacha
        price={200}
        message="Hello Spacha from React!"
    /> : <div>}
</div>
```

:::
